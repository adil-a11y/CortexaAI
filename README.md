 🚀 Cortexa AI – Full Stack AI Agent

Cortexa AI is a full-stack AI-powered application that enables users to interact with an intelligent chatbot using advanced AI models. It includes authentication, payment integration, and a scalable backend architecture.

## 📌 Features

- 🤖 AI Chat powered by Hugging Face models  
- 🔐 User Authentication (JWT-based)  
- 💳 Stripe Payment Integration  
- 📊 User Profile Management  
- ⚡ REST API with Express.js  
- 🌐 Full Stack Architecture (Frontend + Backend)  
- ☁️ Ready for Deployment (Render / Vercel)  


## 🏗️ Tech Stack

### 🔹 Backend
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- JWT Authentication  
- Stripe API  
- Hugging Face Inference API  

### 🔹 Frontend
- React.js  
- Axios / Fetch API  
- Tailwind CSS (if used)
